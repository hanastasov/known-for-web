About

Known For is a web application developed by Jonathan Sharpe, based on an idea by Alex Glossop. For notes on progress, see this GitHub issue.

How to Play

The game is very simple; you are given a randomly-selected actor, and have to guess which three movies they are "known for". If you get the answer right, the title will be shown and the movie's poster is un-blurred. If you don't have any idea, click Skip to pick a new actor.

Data Source

Actor and movie data, including profile and poster images, are supplied by TMDb.

Technology

The Known For front-end is built in Angular 2. It's written using TypeScript, a typed superset of JavaScript.

The back-end API is implemented in Python using aTMDb, a lightweight, asynchronous wrapper around the TMDb API.
